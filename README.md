# Syllabus AI Scanner 

An AI powered web application that converts syllabus assignments into structured daily tasks. Helps students
who dont know how to manage time for big assignments/projects. 

## Features 
- Upload Syllabus or assignment
- AI generated task breakdown
- Calander scheduling

  ## Tech Stack
  - Next.js
  - Typescript
  - Tailwind CSS
